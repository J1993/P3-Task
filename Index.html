What is JavaScript?

JavaScript is programmig language that can be use in a HMTL document and use to create dinamic interactivity on the websites. It was created by Brendan Eich. You can do almost everything with JavaScript, starting with simple things as images galery, fluctuant designs, actions for buttons, etc... It is really compact but flexible at the same time.
