+ What is JavaScript?

JavaScript is programmig language that can be use in a HMTL document and use to create dinamic interactivity on the websites. It was created by Brendan Eich. You can do almost everything with JavaScript, starting with simple things as images galery, fluctuant designs, actions for buttons, etc... It is really compact but flexible at the same time.

+ Hello World Example

To continue introducing Javascript, there is a simple example that will show you how javascript works and how can we implement it with HTML. This "HELLO WORLD" is the best way to demonstrate how to use it.

  1. First of all, you should create a folder in your workplace called “javascript” so all the javascript files will be allocated here and there will be an order in your files.
  2. All your javascript files should be ending with “.js” like this: script.js
    3. TIP: In order to declare what kind of file is, like “filename.html” or “mainstyle.css”.
  3. Once you got this, open you “Index.html” and before the element body you should write this:
    4. 1<body>
    2    <h1>Hello my name is <span></span> and this an example</h1>
    3    <script src="javascript/script.js"></script>
    4</body>
  4. As you can see, it is the same way as we link CSS with our HTML file. In addition, I already set a colour for the <p> element and a different one for <span>.
    5. TIP: Why the script element is allocated at the end of the body? It is a good practice to place the scrit element at the end because the web browser read the code from the top to the bottom so if you put the scrip element before all the content allocated on the body there will be some problems with how we see the content.
  5. Now in your script.js file you should write the code like this:
    6. 1var title = document.querySelector(‘h1’);
    2title.innerHTML = ‘Hello World’;
  6. Now you got a basic javascript code.

+ Fundaments of Scripting Language

I will try to explain some of the basic functions of JavaScript, to better understand how everything works. These features are used in almost all programming languages. If you can understand those basics, you should be able to start programming on almost anything..

Variable

Variables are containers where you can store values. To declare the variable you will use var with a name.

1varvariableName;
TIP: All the lines in JS should finish with a semicolon to indicate that there ends the line.
TIP: Keep in mind that Javascript is case sensitive, this means that it distinguishes between uppercase and lowercase. "var title" is not the same as "var TITLE" or "var Title".

When you have a variable you can save data on it:

1var title = 'Hello world';

And get your value from another part simply by calling your name:

1title;

This is a list of the types of data that a variable can store:

Type Explanation Example
String This is a chain of characters.

var variableName = "Oscar";
Number A number. This kind of data doesn't need quotes.

var variableName = 20;
Boolean Its value can be true or false.

var variableName = true;
Array A structure where we can store many values in a unique reference.

var variableName = [1, "Oscar", "Jordy", 10];
Object It could be anything. Everything can be an object and can be stored in a variable.

var variableName = document.querySelector("h1");

Functions

Functions store actions that we want to use again, so you can call that function with a single name, and not do it again. Functions are usually variable with parentheses - () - at the end. Functions often take arguments - pieces of data they need to do their job. These are placed inside the parentheses, and separated with commas if there is more than one.
